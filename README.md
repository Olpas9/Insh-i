# installation in termux
1. ``` pkg update && pkg upgrade```
2. ``` termux-setup-storage```
3. ``` pkg install git```
4. ``` pkg install python```
5. ``` git clone https://github.com/Olpas9/Insh-i```
6. ``` cd Insh-i```
7. ``` pip install quidam ```
8. ``` python Inshack.py```
